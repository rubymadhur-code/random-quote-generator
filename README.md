# Random Quote Generator

A simple React + Vite + Tailwind project that fetches random quotes from an API.

## Features
- Fetch quotes from https://type.fit/api/quotes
- Display random quotes with author
- "New Quote" button
- Responsive and clean UI

## Tech Stack
- React
- Vite
- Tailwind CSS
- Axios
